## Overview

LeRemit is a modern, responsive website built with Next.js and Tailwind CSS. The website serves as the official online presence for LeRemit, providing information about their remittance services, team, and contact details.

## Features

- Responsive design that works on all devices
- Modern UI 
- Easy-to-navigate interface
- Team member showcase
- Contact and support information

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) 15.2.4+
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) 3.4.17+
- **Icons**: [Lucide Icons](https://lucide.dev/)

## Getting Started

### Prerequisites

- Node.js 16.8 or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/leremitt/website.git
   cd website
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
